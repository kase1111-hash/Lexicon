## Getting Help

### GitHub Issues

For bugs and feature requests, please use GitHub Issues:

- **[Report a Bug](../../issues/new?template=bug_report.md)** - Something isn't working as expected
- **[Request a Feature](../../issues/new?template=feature_request.md)** - Suggest an improvement

Before opening an issue:
1. Search existing issues to avoid duplicates
2. Include relevant details (version, environment, steps to reproduce)
3. Use the provided issue templates

### GitHub Discussions

For questions, ideas, and community discussions, use GitHub Discussions:

- **Q&A** - Ask questions and get answers from the community
- **Ideas** - Share ideas for improvements
- **Show and Tell** - Share projects built with this tool

## Contributing

Interested in contributing? See our [Contributing Guide](CONTRIBUTING.md) for:

## Security Issues

**Do not report security vulnerabilities through public GitHub issues.**

Please see our [Security Policy](SECURITY.md) for instructions on reporting security vulnerabilities responsibly.

## Commercial Support

For commercial support inquiries, please contact the maintainers directly through the repository.

## Community Guidelines

When seeking help or participating in discussions:

- Be respectful and constructive
- Provide context and relevant details
- Search existing resources before asking
- Help others when you can

## Response Times

This is an open-source project maintained by volunteers. Response times may vary:

- **Critical bugs**: We aim to respond within 48 hours
- **Other issues**: Response within 1-2 weeks
- **Feature requests**: Reviewed periodically

Thank you for using Computational Linguistic Stratigraphy!
